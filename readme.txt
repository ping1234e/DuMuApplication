度目门禁客户端使用说明
【背景】
公司使用百度度目考勤机打卡，最近反馈考勤失败。排查发现乘风平台真停服了（之前发公告2025年12月停服），所以才有了本软件。
主要之前人员的注册和同步是通过乘风平台维护的，现在需要一个工具来维护人员信息并推送到设备上。这个软件就是为了解决这个问题而开发的，功能比较简单，主要是维护人员信息和推送到设备上（目前只支持推送人员编号，后续可能会增加其他功能）
【系统要求】
- Windows 7/10/11 或 MacOS 或 Linux
- 已安装 JDK 11 或更高版本

【首次使用】
1. 双击运行 DumuClient.exe（或 DumuClient.jar）
2. 点击"连接设备"按钮
3. 输入门禁机的 IP 地址、端口(默认8080)、密码
4. 点击"保存配置"保存设置

【配置文件说明】
config.properties 文件内容：
device.ip=192.168.1.100    # 设备IP地址
device.port=8080            # 设备端口
device.password=123456      # 设备密码

【功能说明】
- 人员管理：查询、新增、编辑、删除人员信息
- 一键导出：导出人员列表为 CSV 文件
- 一键导入：通过 CSV 文件批量更新人员编号
- 识别记录：查看设备推送的识别记录

【常见问题】
Q: 连接失败怎么办？
A: 检查设备IP是否正确，设备是否在线，密码是否正确

Q: 新增人员失败？
A: 确保人员ID唯一，图片格式为JPG且分辨率在160*160~1080*1080之间
